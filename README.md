# 🧩 SlidePuzzle — Пятнашки

Мобильная игра-головоломка "Пятнашки" для Android с расширенным функционалом: коллекционные карточки, соревновательные режимы, система друзей и ИИ-соперник.

Построена на **Kotlin** и **Jetpack Compose** с архитектурой **MVVM**.

## 📱 О проекте

Классическая головоломка "пятнашки" в современном исполнении — с несколькими режимами игры, системой прогресса и социальными функциями. Проект разрабатывался как учебный/портфолио-проект с акцентом на чистую архитектуру и полноценный пользовательский опыт.

## ✨ Возможности

- **Несколько режимов игры:**
  - Classic — классические пятнашки с цифрами
  - Colors — режим с цветовыми паттернами
  - Images — сборка изображений из частей
  - Time Attack — режим на скорость с выбором подтипа (Classic/Colors/Images)
- **Разные размеры поля** — от 3×3 до 8×8
- **Система карточек-наград** — три уровня редкости (бронза, серебро, золото), выпадающих в зависимости от размера решённого поля
- **Друзья через QR-код** — добавление друзей сканированием QR-кода (на базе ZXing)
- **ИИ-соперник** — генерация оппонента через Anthropic API для соревновательных режимов
- **Аутентификация** — регистрация, вход, редактирование профиля, смена пароля
- **Статистика и достижения** — история игр, таблица лидеров, система ачивок
- **Локализация** — поддержка русского и английского языков
- **Тёмная/светлая тема**

## 🛠️ Технологии

- **Язык:** Kotlin
- **UI:** Jetpack Compose
- **Архитектура:** MVVM (ViewModel, State)
- **База данных:** SQLite (через собственный DatabaseHelper)
- **Хранение настроек:** SharedPreferences
- **QR-коды:** ZXing
- **ИИ:** Anthropic API

## 📋 Требования

- Android Studio (актуальная версия)
- Android SDK
- Минимальная версия Android — уточнить в `build.gradle.kts`
